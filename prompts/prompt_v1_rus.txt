Выступи в роли эксперта по машинному обучению и информационной безопасности.
Твоя задача — написать скрипт на Python для классификации сетевого трафика IoT (выявление кибератак, таких как DDoS, DoS, Recon, и нормального BenignTraffic).

Для загрузки набора данных обязательно используй следующий готовый фрагмент кода:

import kagglehub
# Download latest version
path = kagglehub.dataset_download("subhajournal/iotintrusion")
print("Path to dataset files:", path)

Требования к коду:
1. Скрипт должен найти скачанный CSV-файл в директории `path`.
2. Выполни базовую предобработку: удаление пустых значений.
3. Оптимизированно закодируй категориальные признаки (используй LabelEncoder, чтобы избежать переполнения памяти, так как датасет большой).
4. Раздели данные на обучающую и тестовую выборки.
5. Обучи модель машинного обучения. Выбери алгоритм, который хорошо справляется с объемными табличными данными (например, градиентный бустинг).
6. Выведи метрики качества: Accuracy и classification_report.
