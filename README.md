# Анализ сетевого трафика IoT с использованием LLM

Данный репозиторий содержит результаты командного проекта, выполненного в рамках курса «Безопасность систем Интернета вещей и киберфизических систем» (Весна 2026). 

Главная цель проекта — продемонстрировать возможности использования больших языковых моделей (LLM) для генерации программного обеспечения, анализирующего сетевой трафик и выявляющего кибератаки в IoT-сетях.

## Описание проекта

В рамках работы проведено сравнение классической модели машинного обучения, написанной вручную (Baseline), с моделями, сгенерированными с помощью LLM по специальным промптам. 

Эксперимент подтвердил, что основной массив сетевых аномалий выявляется инструментами, сгенерированными LLM, стабильно и с высокой точностью, однако для работы с редкими векторами атак требуется дополнительная балансировка классов.

* **Набор данных:** [IoT Intrusion Dataset](https://www.kaggle.com/datasets/subhajournal/iotintrusion) (загружается автоматически через API Kaggle).
* **Используемые модели:**
    * `Baseline Model` (Ручная разработка): Random Forest Classifier
    * `LLM Model 1`: Hist Gradient Boosting Classifier
    * `LLM Model 2`: Multi-Layer Perceptron (Нейронная сеть)

## Структура репозитория

* `docs/` — сопроводительная документация (технический отчет и презентация).
* `prompts/` — русскоязычные и англоязычные промпты, использованные для генерации кода.
* `src/hand_coded_models/` — исходный код базовой модели, написанной вручную.
* `src/llm_generated_models/` — исходный код моделей, сгенерированных LLM.
* `requirements.txt` — зависимости проекта.

## Инструкция по запуску

### 1. Подготовка окружения
Клонируйте репозиторий и установите необходимые зависимости:

```bash
git clone <ссылка_на_ваш_репозиторий>
cd iot-traffic-llm-analysis
pip install -r requirements.txt
```
*(Основные зависимости: `kagglehub`, `pandas`, `scikit-learn`)*

### 2. Запуск тестирования
При первом запуске любого из скриптов датасет будет автоматически скачан в кэш вашей системы.

Запуск базовой модели (Random Forest):
```bash
python src/hand_coded_models/baseline_model.py
```
Запуск первой модели от LLM (Градиентный бустинг):
```bash
python src/llm_generated_models/model_llm_1.py
```
Запуск второй модели от LLM (Многослойный персептрон):
```bash
python src/llm_generated_models/model_llm_2.py
```
## Краткие результаты тестирования

На тестовой выборке (~209 тыс. записей) получены следующие метрики Accuracy:

* **Baseline (Random Forest):** 99.30%
* **LLM Model 1 (HistGradientBoosting):** 96.57%
* **LLM Model 2 (MLP Classifier):** 98.49%

Подробный анализ метрик (включая F1-score для различных классов атак) представлен в файле `docs/technical_report.md`.
